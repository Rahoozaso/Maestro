Execution Plan: A series of steps (instructions) generated by the Architect.

Note: The Architect might use actions like ADD_TEST_FILE, REFACTOR, REPLACE_FUNCTION_BODY, IMPLEMENT_FUNCTION, etc. You must interpret these intelligently.

TASK
Apply the changes described in the Execution Plan to the codebase.

DO NOT return a diff or patch yet. Return the full python content.
